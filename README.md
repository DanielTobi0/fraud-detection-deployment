# Fraud Detection Deployment

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](LICENSE)
[![Python](https://img.shields.io/badge/python-3.12%2B-blue.svg)](https://www.python.org/)

End-to-end fraud detection system built on the [PaySim synthetic dataset](https://www.kaggle.com/datasets/ealaxi/paysim1).

**Features:**
- Sequence-aware feature engineering on transaction histories
- XGBoost classifier with class-imbalance handling
- MLflow experiment tracking
- FastAPI prediction service
- Lightweight prediction-score drift monitor

A companion React frontend lives at [fraud-detection-frontend](https://github.com/your-username/fraud-detection-frontend).

## Dataset

This project uses the **PaySim** synthetic financial transactions dataset.

1. Download it from Kaggle: <https://www.kaggle.com/datasets/ealaxi/paysim1>
2. Place the CSV at:
   ```
   data/raw/PS_20174392719_1491204439457_log.csv
   ```

> The raw data file is excluded from version control via `.gitignore` because it is a large (>470 MB) third-party dataset.

## Requirements

- Python 3.12+
- [uv](https://docs.astral.sh/uv/) (recommended) **or** pip

---

## 1) Setup

Always activate the environment before running commands:

```bash
source .venv/bin/activate
```

Install dependencies with uv:

```bash
uv sync
uv sync --group dev
```

## 2) Train the model

```bash
source .venv/bin/activate
python -m src.train
```

Quick smoke run on a subset:

```bash
source .venv/bin/activate
python -m src.train --max-rows 100000
```

Artifacts created:
- `models/fraud_model.joblib`
- `models/feature_columns.joblib`
- `models/reference_stats.joblib`
- `data/processed/train_features.parquet`
- `data/processed/valid_features.parquet`
- `mlruns/` (MLflow local tracking)

## 3) Run the API

```bash
source .venv/bin/activate
uvicorn src.api:app --reload --host 0.0.0.0 --port 8000
```

Health check:

```bash
curl http://127.0.0.1:8000/health
```

Prediction example:

```bash
curl -X POST http://127.0.0.1:8000/predict \
	-H "Content-Type: application/json" \
	-d '{
		"step": 12,
		"type": "TRANSFER",
		"amount": 9834.2,
		"oldbalanceOrg": 12000.0,
		"newbalanceOrig": 2165.8,
		"oldbalanceDest": 500.0,
		"newbalanceDest": 10334.2,
		"orig_txn_count_before": 8,
		"orig_amount_rolling3": 5200.5,
		"orig_step_gap": 1,
		"dest_txn_count_before": 3
	}'
```

Drift status:

```bash
curl http://127.0.0.1:8000/metrics
```

## 4) Run tests and lint

```bash
source .venv/bin/activate
pytest
ruff check .
```
